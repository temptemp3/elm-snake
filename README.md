# elm-snake
Elm implementation of classic game "Snake". A running [demo is available][demo].

For Development:
```
elm reactor
```

To Build:
```
elm make src/Snake.elm --optimize --output=snake.js
```

[demo]: http://martinsnyder.net/projects/snake.html
